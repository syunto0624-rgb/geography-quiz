# 都道府県クイズ

## プロジェクト概要
- **名前**: 都道府県クイズ
- **目標**: 都市名や名産品から都道府県を当てるクイズゲーム
- **機能**: 
  - 47都道府県のクイズデータ
  - 都市名、名産品などの4つのヒント表示
  - 難易度選択（かんたん、ふつう、むずかしい）
  - 正解数・不正解数・正答率の統計表示
  - レスポンシブデザイン

## 完成した機能
- ✅ クイズデータベース（全47都道府県）
- ✅ ランダムクイズ取得API
- ✅ 難易度フィルター機能
- ✅ リアルタイム統計表示
- ✅ アニメーション効果
- ✅ レスポンシブUI

## API エンドポイント
- `GET /api/quiz/random?difficulty={all|easy|medium|hard}` - ランダムなクイズを取得
- `GET /api/prefectures` - 全都道府県リストを取得

## データ構造
- **クイズデータ**: 都道府県名、4つのヒント（都市名、名産品など）、難易度
- **統計データ**: 正解数、不正解数、正答率

## 使い方
1. 難易度を選択（デフォルト: すべて）
2. 表示される4つのヒントを見る
3. 都道府県名を入力して回答
4. 正解・不正解の結果を確認
5. 次の問題へ進む
